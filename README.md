## Псевдо компилятор условного языка в nasm
### Синтаксис
нет ';' в конце выражения
```
a = 1
```

Пробелы между переменными и операторами обязательны
```
(a + b)
```

{} - всегда на отдельной строчке
после if, while выражений
```
if (a > b)
{
a = 1
}
```
```
while a < b
{
a = a - 1
}
```
### Run
1) в начальной папке:
  ```
  zig build run
  ```
  
2) в папке asm:
  ```
  ./run.sh output
  ```
